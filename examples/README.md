```markdown
# exemplos de entrada - nfp-generator

Neste diretório há um exemplo com o formato de entrada esperado pelo programa:

Formato:
- Primeiro: número de vértices do primeiro polígono (inteiro)
- Em seguida: N linhas com "x y" (coordenadas)
- Depois: número de vértices do segundo polígono
- Em seguida: M linhas com "x y"

Arquivo de exemplo: polygons_example.txt

Como usar:
1. Compile o programa (veja build.sh ou os comandos no README principal).
2. Execute o binário e copie/cole o conteúdo de polygons_example.txt quando o programa pedir os vértices,
   ou abra o arquivo e forneça manualmente os valores no terminal conforme solicitado.
```
